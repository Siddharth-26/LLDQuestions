Requirements :
1. Users should be able to select a pickup location a start time and an end time and then they should be able to get the list of all the cars available at that pickup location for that time.
2. The user should be able to book Cars for a duration. They should be able to update the booking or cancel that booking as well.
3. There should not be a scenario of double booking or overlapping bookings for the same car. That should be avoided.
4. Concurrent bookings should be handled properly.
5. Inventory of a store is limited to that store only, that means that a user can book from a pickup location and has to return the car at the same pickup location.

Core Entities:
1. Car
2. Store
3. Bookings
4. Slot

Other Classes:
1. BookingService
2. Orchestrator
3. CarInventory
4. StoreInventory
5. AvailabilityChecker (Not needed Now.)
6. BookingRepo

Flow  :
1. The cars are containing information about cars and the list of objects each object is a Slot these are used to check the availability.
2. The Store contain the information about the store and an inventory of cars belonging to that store.
3. The booking service exposes methods like getAvailableCars(storeId, startTime, endTime) - List<Cars> also exposes methods like bookCar(carId, startTime, endTime) updateBooking(bookingId, startTime, endTime) cancelBooking(bookingId, startTime, endTime)
4. There is an AvailabilityChecker class as well this class is present inside the booking manager and will take care of checking the availability of a particular car or checking the availability of that car.

Class Design :
Car : carId, storeId, carName, carModel, .. List<Slot> bookedSlots; checkOverlap(Slot slot) - boolean ; synchronised addSlot(Slot slot) - boolean; synchronised removeSlot(Slot slot);
Store : storeId, storeName, CarInventory getAvailableCars(startTime, endTime);
BookingService : BookingRepo, StoreInventory, CarInventory, getAvailableCars(storeId, startTime, endTime),  bookCar(carId, startTime, endTime), updateBooking(bookingId, startTime, endTime), cancelBooking(bookingId, startTime, endTime)
Booking : carId, storeId, Slot, bookingId
BookingRepo : HashMap<bookingId, Booking> getBooking addBooking updateBooking deleteBooking
CarInventory : HashMap<CarId, Car> getCar,
StoreInventory : HashMap<StoreId, Store> getStore
PriceCalculator : calculatePrice(Slot, carId);
PaymentService : pay(double amount);


public Booking bookCar(carId, startTime, endTime) throws BookingInterruptedException{
    Slot slot = new Slot(startTime, endTime);
    double price = this.priceCalculator.calculatePrice(Slot, carId);
    if(this.paymentService.pay(price)){
        if(this.carInventory.containsKey(carId){
            Car car = this.carInventory.getCar(carId);
            boolean isAdded = car.addSlot(slot);
            if(isAdded){
                Booking booking = new Booking(carId, car.getStoreId(), slot);
                return booking;
            }
            else{
                throw new BookingInterruptedException("Overlapping slots are already booked.");
            }
            }
        else{
            throw new BookingInterruptedException("Invalid Car Id.");
        }
    }
    }
    else{
        throw new BookingInterruptedException("Payment Could not be processed.");
    }
}

public Booking updateBooking(BookingId, startTime, endTime) throws UpdateBookingException{
    Booking booking = this.bookingRepo.getBooking(bookingId);
    Slot slot = new Slot(startTime, endTime);
    Car car = this.carInventory.getCar(booking.getCarId());
    if(this.bookingRepo.containsKey(bookingId)){
        if(car.checkOverLap(slot)){
            throw new UpdateBookingException("Cannot update booking because there is an overlap between these bookings.");
        }
        else{
            car.removeSlot(booking.getSlot());
            boolean isAdded =  car.addSlot(slot);
            if(isAdded){
                this.bookingRepo.remove(bookingId);
                this.booking.setSlot(slot);
                this.bookingRepo.put(booking.getId(), booking);
                return booking;
            }
            else{
                throw new UpdateBookingException("Cannot update the booking seems like the car is already overallped with other booking for the similar times.");
            }
        }
    }
    else{
        throw new UpdateBookingException("The booking that you are trying to modify is already deleted.")
    }
}

public class Slot implements Comparable<Slot>{
    private DateTime startTime;
    private DateTime endTime;
    private int compareTo(Slot other){
            if(this.startTime.before(other.startTime)){
                return -1;
            }
            else if(this.startTime.after(other.startTime)){
                return 1;
            }
            else{
                if(this.endTime.before(other.endTime)){
                    return -1;
                }
                else if(this.endTime.after(other.endTime)){
                    return 1;
                }
                else{
                    return 0;
                }
            }
}
public synchronised boolean checkOverlap(Slot slot){
    ArrayList<Slot> copy = new ArrayList<>(this.bookedSlot);
    copy.add(slot);
    copy.sort()
    for(Slot currSlot: this.bookedSlots){
        if(currSlot.endTime.before(slot.startTime)){
            continue;
        }
        else if(currSlot.startTime.after(slot.endTime)){
            continue;
        }
        else{
            return false;
        }
    }
    return true;
}

public synchronised boolean addSlot(Slot slot){
    // check if any slot is actually overlapping if yes then return false
    // else add the slot and then return true;
    ArrayList<Slot> copy = new ArrayList<>(this.bookedSlot);
    copy.add(slot);
    copy.sort()
    for(Slot currSlot: this.bookedSlots){
        if(currSlot.endTime.before(slot.startTime)){
                continue;
        }
            else if(currSlot.startTime.after(slot.endTime)){
                continue;
            }
            else{
                return false;
            }
        }
        this.bookedSlot.add(slot);
        this.bookedSlot.sort();
        return true;
}
