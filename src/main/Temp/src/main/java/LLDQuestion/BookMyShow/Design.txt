Requirements :
User should be able to search for movies and then get a list of theatres.
Once the user chooses a theatre then they should be able to see the shows in that theatre.
Once the user clicks on the shows the user should be able to see the seats.
THe user should be able to select the seats and book them and get the bookingId.
Users should be able to cancel the booking.

NFR :
No double bookings
Search should be fast.
